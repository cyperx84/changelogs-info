## Features

### рҹ“Ӣ Release Tracking
Every GitHub release for all 22 harnesses, synced daily. Version history, changelogs, and release notes in one place.

### рҹ“– Cheatsheets
Comprehensive quick-reference guides for 10 harnesses вҖ” commands, flags, config options, workflows, and tips. The Claude Code cheatsheet alone is 500+ lines.

### рҹ§  Model Directory
Track which AI models power these harnesses. Anthropic, OpenAI, Google, Meta, Mistral, and Qwen вҖ” with release dates, context windows, and type classifications.

### вҡҷпёҸ Starter Configs
Real, production-quality configs you can copy-paste. CLAUDE.md templates, .cursorrules, aider configs, OpenClaw gateway setups, and more. Not toy examples вҖ” configs people actually use.

### рҹ”„ Auto-Updating Cheatsheets
Weekly automation pulls from official docs and opens PRs when CLI commands or features change. Never stale.

## Design
