# Cline Cheat Sheet

## Core Workflow
| Step | Action |
|---|---|
| 1 | Open Cline panel in VS Code |
| 2 | Describe task with clear constraints |
| 3 | Review proposed file edits |
| 4 | Approve / reject per action |

## High-Value Tips
- Give Cline exact file paths when possible.
- Break large tasks into small milestones.
- Keep a project rules file for consistency.
