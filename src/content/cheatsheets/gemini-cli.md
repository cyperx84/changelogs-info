# Gemini CLI Cheat Sheet

## Core Commands
| Command | What it does |
|---|---|
| `gemini` | Start interactive session |
| `gemini "prompt"` | One-shot prompt |
| `gemini -f file.ts "explain"` | Include file context |

## Useful Flags
| Flag | Purpose |
|---|---|
| `--model <name>` | Choose model |
| `--json` | JSON output |
| `--temperature <n>` | Creativity control |
| `--help` | Show help |

## Tips
- Keep prompts specific and code-focused.
- Use one-shot mode for scripts and automation.
